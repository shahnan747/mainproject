# FieldHub Backend

This is the backend of **FieldHub**, a MERN-based field sales and order management system. It provides REST APIs for authentication, store and product management, order processing, delivery management, analytics, AI chatbot integration, and email notifications.

## Technologies Used

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT (JSON Web Token)
* Nodemailer
* CORS
* dotenv
